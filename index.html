Caribu

Tech Stack:

Typescript, React, Redux, Express, Electron, Babel

Caribu assists with the migration of a program from monolith to microservice. Through the act of parsing the abstract syntax tree of an express application, it can extract the endpoints, functions, and dependencies and visualize how isolated a function is. The intended goal of the visualization is to streamline the process of figuring out what functions can be broken down and seperated into AWS Lambda functions. The program shows groupings of dependencies, functions, and endpoints with code snippets even showing the line in code where they first appear.

Arbit

Tech Stack:

React, Express, MongoDB

Arbit is a full stack application that serves as a framework to calculate and store arbitrages. It leverages prices yielded from a currency API developed in partnership with Amazon's currency exchange API and uses personally built trading algorithms to present the most profitable path of trade to yield profit.

Photogenicus

Tech Stack:

TensorFlow, Javascript, React, Express, BetterSQLite3

Photogenicus is tool for live identity validation of an individual based off of historic images. By uploading a set of facial images, a TensorFlow model can be generated that can confirm the presence of a persons face in the image set. Leveraging that functionality, we can assert whether the person in a camera is within the personnel database and approve or deny access to entry.

Let's Arcade

Tech Stack:

Python, Flask, Axios, Javascript, Vue, Express, Websocket, MongoDB

Lets arcade was a community based gaming application developed under Techben. The infrastructure and nature of the site favored MongoDB. Features included user accounts, a selection menu for games, and game rooms to play as well as chat in.
